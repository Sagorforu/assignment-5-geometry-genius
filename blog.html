Question Number 1

What is DOM and what is the purpose of DOM?

Answer: The document object model called DOM. It is a programming interface for web documents. It represents the page so that we can style the document structure, styled tag, content, and etc. Programming languages can interact with the page using DOM as nodes and objects. All of the properties, methods, and events available for styling, changing, manipulating and creating web pages are organized and stylish into objects. We can easily get access to style, change, manipulate etc to change something awesome. That's why DOM is invented.

Question Number 2

How will you select HTML elements using DOM? Name the DOM methods.

Answer: There are few ways to Connect DOM into HTML tag or objects. DOM selects HTML elements within a document using JavaScript below the ways:

  1. 1. getElementsByTagName()

    In this way you can add html tag and it will select html tag directly.

  2. 2. getElementsByClassName()

    In this way you can add by class name and it will select that tag which is mentioned by the class name. It's very useful.

  3. 3. getElementById()

    In this way you can add by Id name and it will select that tag which is mentioned by the Id name. It's also very useful and important.

  4. 4. querySelector()

    In this way you can add by specific css selector and it will select that tag which is select first element that matches the specified CSS selector in the document.

  5. 5. querySelectorAll()

    In this way you can add by specific css selector and it will select that tag which is select all element that matches the specified CSS selector in the document.

Question Number 3

What is event bubble?

Answer: Actually, Event bubble is the part of DOM. It's relate to DOM concept. It' working when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element. We can understand it using ParentNode or ChildNode. It's looks like a tree which is bubble parent to child. When you click the button, you can think of it like you're also clicking the child of that tag. This is because the button is a child of the that tag.

Question Number 4

What is a callback function and why will you use it?

Answer: The function which executed after another function has finished execution. This function passed as an argument to another function. That's why it's called a callback function.
Callback function needed sometime. Mostly, when many JavaScript actions are asynchronous. This function make sure us that is not going to run before a code running is completed but will run right after the code executed. A callback function purpose is to execute code in response to an event while the rest of the code runs.
